How AI Testing Tools Works

  1. Record — Click Start to record browser interactions. Clicks, typing, navigation, and assertions are captured automatically with smart selectors.
  2. AI Generate — Describe a test scenario in plain English and AI generates production-ready Playwright test code with Page Object Model support.
  3. Tests — Run your test suites against different environments via GitHub Actions with a single click.
  4. Suites — Manage saved tests, export individual .spec.ts files, or download all as a batch.

Keyboard Shortcuts

  • Ctrl + Shift + R — Start/stop recording
  • Ctrl + Shift + E — Export current recording
  • Esc — Close preview modal
